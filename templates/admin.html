🍪 B站Cookie配置

配置B站Cookie后，可以获取评论数据。Cookie需要从浏览器获取。

💻 获取Cookie步骤

  1. 电脑浏览器打开 bilibili.com 并登录账号
  2. 按 F12 打开开发者工具
  3. 点击顶部 Network（网络）标签
  4. 在浏览器地址栏按 回车 刷新页面
  5. 在左侧列表点击任意一个请求名称
  6. 在右侧找到 Headers → Request Headers → Cookie
  7. 复制整个Cookie值，粘贴到下方输入框
DEDE_USER_ID=xxx; SESSDATA=xxx; bili_jct=xxx...
